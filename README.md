# Home SOC Lab

## Overview

This repository documents my journey learning Security Operations Center (SOC) analysis through hands-on lab exercises.

The lab is designed to simulate a real-world SOC environment where I can detect, investigate, and respond to security events.

## Lab Environment

- Host Machine: MacBook Pro 2017
- Virtualization: VirtualBox
- SIEM: Wazuh
- IDS: Suricata
- Attacker Machine: Kali Linux
- Server: Ubuntu Server
